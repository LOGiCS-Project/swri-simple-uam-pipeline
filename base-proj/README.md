A set of libraries and wrappers around the SWRi athens-uav project.  

## Installation

With `pip`:
```bash
pip install simple-uam
```

With [`pipx`](https://github.com/pipxproject/pipx):
```bash
python3.7 -m pip install --user pipx
pipx install simple-uam
```
